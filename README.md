# Emergency Room Visit Analysis Dashboard (Power BI)

This Power BI dashboard presents a detailed analysis of patient visits to the emergency room, offering insights into operational efficiency, patient demographics, satisfaction levels, and departmental referrals. The visualizations are designed to support data-driven decision-making for hospital administrators and healthcare analysts.
